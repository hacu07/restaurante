Acerca de sus servicios

Este software tiene como funcion agilizar el proceso de los pedidos de restaurantes y bares. Dispone de cuatro modulos, los cuales son: Modulo Administrador, Modulo cajero, Modulo Jefe de cocina y modulo Mesero. Hace uso de dispositivos reduciendo el impacto ambiental, sus ventajas son:

  • Reducción del tiempo de espera del pedido
  • Simplificación de procesos actuales
  • Eliminación de errores en el pedido
  • Control total sobre el proceso del pedido
  • Estadísticas de ventas y productos

Funcionalidades

Este software cuenta con varias funcionalidades, las cuales tienen un orden respectivo para poder ser ejecutado:

Asignar mesa y Ordenar pedido

Esta funcion esta a cargo del modulo mesero.

Cambiar estados del pedido

Esta funcion esta a acargo del modulo de cocina, el cual cambia los estados del pedido según en el que se encuentren, una vez sean solicitados por el mesero

Confirmacion del pedido

Esta funcion es realizada en el modulo del mesero; Una vez este listo el producto o pedido, el mesero debera confirmar la entrega de este.

Agregar Productos

Esta funcion la realiza el mesero, puede agregar o quitar productos a un pedido antes. Una vez sea entregado el producto o facturado el pedido, ya no se podra realizar esta funcion.

Facturar pedidos

Esta funcion la realiza el cajero, una vez el pedido sea entregado y el mesero confirme esto, el cajero podra generar la factura del pedido correspondiente y su estado automaticamente cambia

Eliminar o agregar

Esta funcion unicamente la puede realizar el encargado del modulo administrador, puede agregar, cambiar o eliminar empleados, categorias y productos.

Estadisticas

Las estadisticas unicamente las puede mirar el encargado del modulo Administrador, son generadas por productos, mensuales y diarias

Control de los pedidos

El control de los pedidos lo tiene el encargado del modulo administrador, puede ver que pedidos se han realizado, cual es el pedido, su hora y su estado.
